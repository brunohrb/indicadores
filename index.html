Sem dados para os meses selecionados

Rode o sync para um mês específico em GitHub Actions → Sync Power BI com input mes=YYYY-MM. Depois volte aqui e clique em 🔒 Fechar mês para congelar os valores.

Botões: "Ver este mês" mostra só o mês escolhido (substitui o que estava). "+ Comparar" adiciona o mês escolhido aos já visíveis (soma/agrega vários meses).

Como isso funciona: os valores vêm do Power BI via DAX e são gravados no Supabase. O mês 🔄 ao vivo é o último que o sync rodou e pode mudar a cada execução. Quando os números do mês estão consolidados (~dia 5–6 do mês seguinte), clique em 🔒 Fechar mês para criar um snapshot imutável — depois disso, os valores não mudam mais (ideal para fechar comissão).

Vários meses: ao comparar, os cards são agregados — soma para contagens/valores (cancelamentos, novos negócios, receita...), média para tickets médios, e último valor para tamanhos de base (Base PF, PJ, Contratos, Isentos), porque base é snapshot e não acumula.

Atenção: cards marcados com ⚠️ inferido podem ter valores diferentes do Power BI original. Compare e avise os que estiverem errados.
